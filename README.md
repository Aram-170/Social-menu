# Social Share Menu

Un menu circolare interattivo e moderno per la condivisione sui social, realizzato con **HTML5, CSS3 (Flexbox & CSS Variables)** e un pizzico di **JavaScript** per la gestione degli stati.

## ✨ Funzionalità
- **Animazioni Fluide:** Effetto elastico (*cubic-bezier*) all'apertura e alla chiusura.
- **Effetto Neon:** Bagliore dinamico al passaggio del mouse (*hover*) basato sul colore del brand del social.
- **Controrotazione Automatica:** Le icone rimangono perfettamente dritte durante l'espansione circolare.
- **Chiusura Intelligente:** Il menu si chiude automaticamente se si clicca in un punto qualsiasi fuori dal componente.

## 🛠️ Tecnologie Utilizzate
- HTML5
- CSS3 (Custom Properties, Transizioni, Trasformazioni 2D)
- JavaScript (ES6)
- [Ionicons](https://ionic.io/ionicons) per la libreria di icone



# Social Share Menu

An interactive, modern, and fluid circular social sharing menu built using **HTML5, CSS3 (Flexbox & CSS Variables)**, and a touch of **JavaScript** for state management.

## ✨ Features
- **Smooth Animations:** Elastic bounce effect (*cubic-bezier*) on opening and closing states.
- **Neon Glow Effect:** Dynamic glow on mouse hover based on each social network's official brand color.
- **Automatic Counter-Rotation:** Icons remain perfectly upright and readable while expanding in a circle.
- **Smart Close:** The menu automatically closes if you click anywhere outside the component area.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Custom Properties, Transitions, 2D Transformations)
- JavaScript (ES6)
- [Ionicons](https://ionic.io/ionicons) for the premium icon library
